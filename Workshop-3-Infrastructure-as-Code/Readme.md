# Commands
1. init local terraform working environment

    ```bash
    terraform init
    ```

2. Review stack creation plan

    ```bash
    terraform plan
    ```

3. Create stack

    ```bash
    terraform apply
    ```


4. Connect to  Elastic Beanstalk application and environment

    ```bash
    eb init
    ```
5. Deploy application

    ```bash
    eb deploy
    ```
